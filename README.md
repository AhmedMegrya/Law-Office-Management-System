# نظام إدارة مكتب المحاماة

نظام إدارة مكتب المحاماة هو تطبيق مكتبي مبني باستخدام Python و PyQt5، مصمم لمساعدة مكاتب المحاماة في إدارة عملائها، قضاياها، جلساتها، الرسوم، المصروفات، والتوكيلات، بالإضافة إلى نظام أرشفة إلكتروني للمستندات.

## الميزات الرئيسية

*   **إدارة العملاء:** تسجيل وتتبع بيانات العملاء.
*   **إدارة القضايا:** متابعة تفاصيل القضايا، أرقامها، المحاكم، الخصوم، والموضوعات.
*   **إدارة الجلسات:** تسجيل مواعيد الجلسات، القرارات السابقة، والإجراءات المطلوبة.
*   **إدارة الرسوم والمدفوعات:** تتبع الرسوم المستحقة والمدفوعات لكل قضية.
*   **إدارة المصروفات:** تسجيل المصروفات المتعلقة بالقضايا.
*   **إدارة التوكيلات:** حفظ بيانات التوكيلات وتواريخ انتهائها.
*   **الأرشيف الإلكتروني:** رفع وحفظ المستندات المتعلقة بالقضايا والعملاء في مجلدات منظمة.
*   **لوحة تحكم:** عرض ملخص سريع لأهم الإحصائيات والتنبيهات.
*   **دعم اللغة العربية:** واجهة مستخدم باللغة العربية ودعم كامل للبيانات العربية.

## متطلبات التشغيل

للتشغيل من الكود المصدري، تحتاج إلى:

*   Python 3.8
*   المكتبات المذكورة في `requirements.txt`

## البناء والتوزيع (Portable EXE)

يمكن بناء نسخة محمولة (Portable EXE) من التطبيق باستخدام PyInstaller، مما يسمح بتشغيل التطبيق مباشرة دون الحاجة لتثبيت Python أو أي مكتبات على جهاز المستخدم.

للبناء، استخدم ملف `build_exe.bat`.

## هيكل المشروع

```
Law_Office_Management_System/
├── main.py                 # نقطة الدخول الرئيسية للتطبيق
├── login.py                # شاشة تسجيل الدخول
├── main_window.py          # النافذة الرئيسية للتطبيق
├── database.py             # إدارة قاعدة البيانات SQLite
├── styles.py               # تعريف الأنماط (CSS) للتطبيق
├── widgets.py              # عناصر واجهة المستخدم المخصصة
├── requirements.txt        # قائمة المكتبات المطلوبة
├── build_exe.bat           # سكريبت بناء النسخة المحمولة (EXE)
├── law_office.db           # قاعدة البيانات (تُنشأ تلقائيًا إذا لم توجد)
├── Archive/                # مجلد الأرشيف الإلكتروني للمستندات
├── Backups/                # مجلد النسخ الاحتياطي لقاعدة البيانات
├── assets/                 # الأصول مثل الأيقونات والصور
├── pages/                  # وحدات (Modules) صفحات التطبيق المختلفة
│   ├── __init__.py
│   ├── archive.py          # صفحة الأرشيف
│   ├── cases.py            # صفحة القضايا
│   ├── clients.py          # صفحة العملاء
│   ├── dashboard.py        # لوحة التحكم
│   ├── fees.py             # صفحة الرسوم
│   ├── poa.py              # صفحة التوكيلات
│   ├── reports.py          # صفحة التقارير
│   ├── sessions.py         # صفحة الجلسات
│   └── settings.py         # صفحة الإعدادات
└── .gitignore              # ملف لتجاهل الملفات غير المرغوبة في Git
```
