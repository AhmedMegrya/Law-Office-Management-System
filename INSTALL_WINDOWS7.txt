تعليمات تثبيت نظام إدارة مكتب المحاماة على Windows 7 (للمطورين)
=================================================================

هذا الملف مخصص للمطورين أو المستخدمين الذين يرغبون في تشغيل المشروع من الكود المصدري على نظام Windows 7 64-bit، أو إعادة بناء النسخة المحمولة (EXE).

**ملاحظة هامة:** إذا كنت مستخدمًا عاديًا، لا تحتاج إلى اتباع هذه الخطوات. فقط قم بتشغيل ملف `LawOfficeSystem.exe` مباشرة من النسخة المحمولة.

## 1. تثبيت Python 3.8

يجب استخدام إصدار Python 3.8 لضمان أفضل توافق مع المكتبات القديمة المدعومة على Windows 7.

1.  **تحميل Python 3.8:**
    قم بزيارة صفحة الأرشيف الخاصة بـ Python وقم بتحميل المثبت المناسب لـ Windows 7 64-bit:
    [https://www.python.org/downloads/release/python-3810/](https://www.python.org/downloads/release/python-3810/)
    ابحث عن `Windows x86-64 executable installer`.

2.  **تشغيل المثبت:**
    *   قم بتشغيل ملف المثبت الذي قمت بتحميله.
    *   **مهم جدًا:** تأكد من تحديد خيار `Add Python 3.8 to PATH` أثناء عملية التثبيت.
    *   اتبع التعليمات لإكمال التثبيت.

3.  **التحقق من التثبيت:**
    افتح موجه الأوامر (CMD) واكتب:
    ```cmd
    python --version
    pip --version
    ```
    يجب أن تظهر لك إصدارات Python و pip (يجب أن تكون Python 3.8.x).

## 2. إعداد بيئة المشروع

1.  **فك ضغط المشروع:**
    قم بفك ضغط ملف المشروع `Law_Office_Management_System.zip` إلى مجلد من اختيارك (مثال: `C:\LawOfficeSystem`).

2.  **فتح موجه الأوامر في مجلد المشروع:**
    *   اذهب إلى مجلد المشروع الذي فككت ضغطه.
    *   في شريط العنوان بمتصفح الملفات، اكتب `cmd` واضغط Enter لفتح موجه الأوامر في هذا المسار.

3.  **تثبيت المكتبات المطلوبة:**
    داخل موجه الأوامر، قم بتثبيت المكتبات المحددة في `requirements.txt`:
    ```cmd
    pip install -r requirements.txt
    ```
    هذا سيقوم بتثبيت الإصدارات المتوافقة من PyQt5 و reportlab و Pillow و PyInstaller.

## 3. تشغيل المشروع من الكود المصدري

بعد تثبيت Python والمكتبات، يمكنك تشغيل التطبيق مباشرة:

```cmd
python main.py
```

## 4. بناء النسخة المحمولة (EXE)

لإنشاء نسخة محمولة من التطبيق (ملف EXE واحد لا يتطلب تثبيت Python):

1.  **تأكد من إعداد بيئة المشروع** كما هو موضح أعلاه.
2.  **تشغيل سكريبت البناء:**
    داخل موجه الأوامر في مجلد المشروع، قم بتشغيل ملف `build_exe.bat`:
    ```cmd
    build_exe.bat
    ```
    سيقوم هذا السكريبت بتنظيف ملفات البناء القديمة، تثبيت المتطلبات (إذا لم تكن مثبتة)، بناء ملف `LawOfficeSystem.exe` باستخدام PyInstaller، ونسخ الملفات الضرورية (قاعدة البيانات، مجلدات الأرشيف والنسخ الاحتياطي) إلى مجلد `dist\LawOfficeSystem_Portable`.

3.  **النسخة النهائية:**
    ستجد النسخة المحمولة النهائية في المسار `dist\LawOfficeSystem_Portable`.
